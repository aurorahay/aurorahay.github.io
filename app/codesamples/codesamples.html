A simple adapter (JavaScript, _.js, AngularJS)

This code sample was used to convert pretty much any JSON data set to create a D3.js line graph. The main method used is "convertModelToGraphModel". That method is only about 20 lines long, but, there is a whole mess of lodash.js going on to make it look simple.

This adapter made making line graphs a cinch when using large sets of JSON that looked completely different!
